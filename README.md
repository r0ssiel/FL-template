# Forbidden Lands Template
This repository contains a LaTeX template (document class) for making documents similar to the Forbidden Lands books. Don't use this template for comercial purpuses. Its only intended use regards free content made by the community to the community.

# Example file: Cheat Sheet in Portuguese
The example provided in this repository is that of a cheat sheet of Forbidden Lands rules translated to Portuguese. This cheat sheet is mostly the one first conceived by [u/gypsyfreak](https://www.reddit.com/user/gypsyfreak/), which you can still acess [here](https://www.reddit.com/r/rpg/comments/tzlk67/for_any_forbidden_lands_fans_i_made_this_player/).
